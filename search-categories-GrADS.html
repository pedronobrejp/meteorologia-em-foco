Grid Analysis and Display System (GrADS)
O GrADS é uma ferramenta de desktop interativa que é usada para fácil acesso, manipulação e visualização de dados de ciências da terra. O GrADS tem dois modelos de dados para manipulação de dados em grade e de estação. O GrADS suporta muitos formatos de arquivo de dados, incluindo binário (fluxo ou sequencial), GRIB (versões 1 e 2), NetCDF, HDF (versões 4 e 5) e BUFR (para dados de estação).
